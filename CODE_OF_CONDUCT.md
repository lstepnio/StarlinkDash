# Code of Conduct

## Our Standard

Treat contributors and users with respect, patience, and professionalism.

Unacceptable behavior includes:

- harassment
- intimidation
- discriminatory language
- doxxing or sharing private information
- bad-faith or abusive interactions

## Enforcement

Project maintainers may remove comments, issues, pull requests, or participants that violate these expectations.

## Reporting

Use the repository's private reporting path when possible for sensitive behavior concerns.
